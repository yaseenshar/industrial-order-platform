# Industrial Order Management Platform

A standard order management Java Spring Boot microservices project built to practice backend engineering, event-driven architecture, and modern DevOps workflows.

## Goals

- Build production-style Spring Boot microservices
- Practice REST API design
- Use Kafka for asynchronous communication
- Use PostgreSQL for service data storage
- Add Docker-based local development
- Add CI/CD with GitHub Actions
- Improve system design and architecture skills

## Services

- api-gateway
- order-service
- inventory-service
- notification-service
- audit-service

## Tech Stack

- Java 21
- Spring Boot 3
- PostgreSQL
- Kafka
- Docker Compose
- GitHub Actions
- JUnit 5
- Testcontainers
- Prometheus
- Grafana
